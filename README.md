# Skill Matcher Admin Panel

Admin dashboard for managing tests, questions, prompts, rules, and workflows for the Skill Matcher platform. Built with Next.js and MUI.

## Tech stack

- **Framework**: Next.js 14 (React 18)
- **UI**: MUI (Material UI)
- **Charts**: ApexCharts
- **HTTP**: Axios

## Getting started

```bash
# install dependencies
npm install

# start development server
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Scripts

- **`npm run dev`**: Start the development server.
- **`npm run build`**: Create a production build.
- **`npm start`**: Run the production build.
- **`npm run lint`**: Run ESLint checks.

## Environment

Create a `.env` file (based on your local needs) for API URLs and secrets. Do **not** commit real secrets to version control.
